## Project Overview

This project is a **personal resume webpage** designed with a modern layout to present information in a clear and professional way.

The webpage uses a **two-column design**:

- **Left Sidebar** – Personal information, contact details, skills, and interests  
- **Main Section** – Profile summary, education, projects, courses, and strengths

The goal of this project was to practice:

- HTML page structuring  
- CSS styling and layout design  
- Building a simple personal portfolio webpage  

---

- Clean two-column resume layout

- Structured content sections  
- Professional resume presentation
